# Doc Quality Gate Pack

This pack defines the machine-readable contracts, manifests, policies, suites, and waiver model for doc-quality governance.

The repository does not currently expose a wired root command for this pack. Treat it as a reusable policy definition until a real script is added.

Use this pack when the repository needs to answer:

- which docs are governed
- which contract applies to each doc class
- which issues are blocking versus warning-level
- which waivers exist and when they expire
- where the deterministic governance report should be written

The durable rationale should live in repository documentation when this pack is activated.
