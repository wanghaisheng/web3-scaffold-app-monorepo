# Project Readiness Packet Schema

The readiness packet contract checks markdown files for explicit packet fields.

The current `milestone-packet` contract requires:

- goal
- in scope
- out of scope
- dependencies
- interface or delivery list
- acceptance criteria
- validation package
- fallback note
- linked docs or linked change record

The default target set covers the change-record task template plus active `openspec/changes/*/tasks.md` records outside the archive tree.
