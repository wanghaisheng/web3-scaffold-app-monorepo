# `@status` Compatibility Guide

`@status` is a legacy alias page kept for compatibility.

This repository no longer treats status as a separate user-facing mode.

## Local Mapping

Status is now reflected through:

- the current Quick or BMM workflow context
- the active change record under `openspec/changes/<change-name>/`
- the local closeout and validation rules under `.codex/core/`

Use this page only as a reminder that the alias is historical.
