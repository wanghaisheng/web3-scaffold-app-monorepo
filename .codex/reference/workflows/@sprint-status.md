# `@sprint-status` Compatibility Guide

`@sprint-status` is a copied alias from older BMAD sprint wrappers.

In this repository, progress tracking is expressed through:

- the active BMM milestone plan
- the current `openspec/changes/<change-name>/` record
- local validation and closeout rules

Use this page only as historical alias context.
