# Legacy Specs Archive

This folder is an archived copy of the former root `.specs/` tree.

Use it as reference material only. It is no longer the active source of truth for workflow or implementation in this repository.

Prefer these active sources first:

- `AGENTS.md`
- `.codex/`
- `openspec/changes/`
- `openspec/specs/` when long-lived specs are added there

The archived topical groups are preserved under:

- `engineering/`
- `growth/`
- `guides/`
- `product/`
- `protocols/`
- `review/`

When an archived note still refers to `specs/...`, read that as `ref/legacy-specs/...`.
