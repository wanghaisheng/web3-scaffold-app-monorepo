# Topic Cluster 主题集群策略

> Topic Cluster 是现代SEO的核心架构，通过支柱页与集群页的互链结构，建立主题权威性。

## 📋 核心概念

### 什么是 Topic Cluster？

```
                    ┌─────────────┐
                    │   支柱页    │
                    │ (Pillar)   │
                    └──────┬──────┘
                           │
         ┌─────────────────┼─────────────────┐
         │                 │                 │
    ┌────▼────┐       ┌────▼────┐       ┌────▼────┐
    │ 集群页1 │       │ 集群页2 │       │ 集群页3 │
    │(Cluster)│       │(Cluster)│       │(Cluster)│
    └─────────┘       └─────────┘       └─────────┘
         │                 │                 │
    ┌────▼────┐       ┌────▼────┐       ┌────▼────┐
    │ 集群页4 │       │ 集群页5 │       │ 集群页6 │
    └─────────┘       └─────────┘       └─────────┘
```

- **支柱页 (Pillar Page)**: 覆盖主题的全面性页面，锚定核心关键词
- **集群页 (Cluster Page)**: 深入探讨子主题的页面，锚定长尾关键词
- **内链结构**: 所有集群页链接到支柱页，支柱页链接到所有集群页

## 📝 页面类型标准结构

### 支柱页结构模板

```markdown
# [主题]完全指南 | [品牌名]

## 简介（含核心关键词，前100字）

## 目录（可跳转）

## 什么是[主题]？（Informational）

## [主题]的类型（链接到类型对比集群页）

## 如何选择[主题]（链接到选购指南集群页）

## [主题]价格指南（链接到定价集群页）

## 推荐产品/品牌（链接到产品页）

## 常见问题FAQ（Schema标记）

## 总结与下一步（CTA：询价/联系）

## 相关资源（链接到所有集群页）
```

### 集群页结构模板

```markdown
# [具体子主题] | [品牌名]

## 简介（含目标关键词）

## 主要内容

### 子标题1（含长尾关键词）
### 子标题2
### 子标题3

## 与其他主题的关联（内链到支柱页和相关集群页）

## 常见问题FAQ

## 下一步行动（CTA）

## 相关阅读（链接到支柱页 + 2-3个相关集群页）
```

## 🔗 内链策略

### 内链规则

1. **每个集群页必须链接到支柱页**（锚文本使用支柱页核心关键词）
2. **支柱页必须链接到所有集群页**
3. **相关集群页之间互相链接**（每页5-20个内链）
4. **锚文本使用目标页面的关键词**（避免"点击这里"）

### 内链锚文本规范

| 目标页面类型 | 推荐锚文本格式 | 示例 |
|-------------|---------------|------|
| 支柱页 | [主题]完全指南 | CT扫描仪完全指南 |
| 产品对比页 | [A] vs [B] 对比 | 16排 vs 64排CT对比 |
| 定价页 | [产品]价格指南 | CT扫描仪价格指南 |
| 产品页 | [品牌][产品名] | 联影uCT 760 |
| 品牌页 | [品牌名]设备 | 联影医疗设备 |

## 📊 Cluster 发布优先级

### 优先级评估标准

| 因素 | 权重 | 说明 |
|------|------|------|
| 商业价值 | 40% | 是否直接支持Money Pages |
| 搜索量 | 25% | 目标关键词总搜索量 |
| 竞争难度 | 20% | 关键词KD平均值 |
| 内容储备 | 15% | 是否有现成素材 |

### 发布路线图

**第1个月（P0）**
- 完成所有5个支柱页
- 完成CT/MRI定价集群页
- 完成中国vs西方品牌对比页

**第2-3个月（P1）**
- 完成产品类型对比页
- 完成主要品牌介绍页
- 完成进口设备流程指南

**第4-6个月（P2）**
- 完成技术类集群页
- 完成区域案例页
- 完成科普类集群页
