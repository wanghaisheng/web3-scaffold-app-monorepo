## ✅ 提交流程

### 提交前准备

```markdown
准备材料清单:
- [ ] 公司Logo（多种尺寸：200x200, 400x400, 高清原图）
- [ ] 公司简介（50/150/300/500字版本）
- [ ] 关键词列表（5-10个）
- [ ] 联系信息（邮箱、电话、地址）
- [ ] 社交媒体链接
- [ ] 产品/服务图片（3-5张）
```

### 提交步骤

```
1. 创建/登录目录账户
2. 选择合适的类别
3. 填写公司信息
4. 上传Logo和图片
5. 添加网站URL（确保指向正确页面）
6. 提交并等待审核
7. 记录到追踪表格
```

### 提交后跟进

```
审核周期:
- 免费目录: 1-4周
- 付费目录: 1-7天
- 手动审核目录: 2-8周

跟进操作:
- 每周检查提交状态
- 审核通过后验证链接
- 更新追踪表格
- 定期更新信息（每6-12个月）
```

### 状态定义

- ⬜ 待提交: 尚未开始
- 🟡 审核中: 已提交等待审核
- ✅ 已通过: 审核通过，链接有效
- ❌ 已拒绝: 审核未通过
- 🔄 需更新: 信息需要更新

## ⚠️ 注意事项

```
✅ 推荐做法:
- 优先提交高DR目录
- 使用一致的NAP信息（Name, Address, Phone）
- 选择最相关的类别

- 批量提交低质量目录
```
