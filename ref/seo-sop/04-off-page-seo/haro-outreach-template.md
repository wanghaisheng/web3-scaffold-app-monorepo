# HARO回复模板

> 快速、专业的HARO回复是获取高质量媒体外链的关键。

## 📋 HARO基础信息

### 什么是HARO?

HARO (Help A Reporter Out) 是连接记者与行业专家的平台。记者发布采访请求，专家提供见解并获得媒体曝光和外链。

### 订阅设置

**注册链接**: https://www.helpareporter.com/sources/

**推荐订阅类别**:
- Healthcare and Medical
- Technology
- Business and Finance
- International

**邮件接收时间** (EST):
- 早间: 5:35 AM
- 午间: 12:35 PM
- 晚间: 5:35 PM

## 🎯 筛选相关请求

### 关键词过滤

```
优先响应的话题:
- medical imaging / medical equipment
- CT scanner / MRI machine
- healthcare technology
- China manufacturing
- B2B / international trade
- hospital equipment
- radiology
- diagnostic imaging
```

### 快速评估清单

```
✅ 应该回复:
- 主题与医疗设备/中国制造相关
- 明确的发布媒体（DR≥30）
- 截止时间充足（≥24小时）
- 需要行业专家观点

❌ 跳过:
- 完全不相关的话题
- 无名媒体或个人博客
- 截止时间已过或<6小时
- 要求过于具体（如需要特定医院数据）
```

## ✅ 回复最佳实践

### 时间策略

```
⚡ 响应时间优先级:
- 截止<24小时: 立即回复（15分钟内）
- 截止1-3天: 当天回复
- 截止>3天: 24小时内回复

💡 提示: 越早回复，被选中概率越高
```

### 内容质量检查

```markdown
发送前检查:
- [ ] 直接回答了问题
- [ ] 包含具体数据或例子
- [ ] 提供了独特见解（非泛泛而谈）
- [ ] 语法正确，无拼写错误
- [ ] 包含简短的个人/公司介绍
- [ ] 附上联系方式
```

### 跟进策略

```
如果被采用:
1. 等待文章发布
2. 检查是否包含链接
3. 在社交媒体分享
4. 记录到外链追踪表

如果未被采用:
- 不需要跟进询问
- 继续回复其他请求
- 分析是否可以改进回复
```
