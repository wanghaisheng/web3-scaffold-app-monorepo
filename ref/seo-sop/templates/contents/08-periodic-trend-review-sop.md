# 周期性内容更新与趋势响应 SOP

> 内容发布不是终点，而是优化的起点。通过数据驱动的周期性复盘，持续提升内容表现并捕捉新流量机会。

---

## 🔄 核心复盘周期表

| 周期 | 重点任务 | 数据源 | 目标 |
|------|----------|--------|------|
| **每周 (Weekly)** | 社交媒体热点捕捉 | TikTok/IG/Reddit | 发现病毒式话题 (e.g., #NatureOzempic) |
| **每月 (Monthly)** | GSC 表现复盘 | Google Search Console | 优化 "射程内" 关键词，修补流量下跌 |
| **每季 (Quarterly)** | 趋势与季节性规划 | Google Trends | 布局下一季度的季节性内容 |

## 📊 模块 A: GSC 数据挖掘 (月度)

### 1. 寻找 "射程内" 关键词 (Striking Distance Keywords)
这些关键词排名在 **4-20** 名之间，稍加优化即可进入前 3 或首页。

**操作步骤**:
1. 打开 GSC > 搜索结果 > 日期范围选 "过去 28 天"。
2. 筛选位置 (Position): 大于 3 且 小于 20。
3. 按展示量 (Impressions) 降序排列。
4. **行动**:
   - 检查对应页面是否充分覆盖了该关键词。
   - 如果没有，**添加一个新的 H2 章节** 专门讨论该话题。
   - 如果已有，**优化现有段落**，增加深度或 FAQ。

### 2. 发现新搜索意图 (New Queries)
用户可能会用你未预料到的词搜索你的页面。

**操作步骤**:
1. 选中某个高流量页面 (Page)。
2. 查看该页面的查询词 (Queries)。
3. 寻找内容中**未明确提及**但带来展示/点击的词。
4. **行动**: 将这些新词作为新的 H2 或 FAQ 添加到页面中。

### 3. 诊断流量下跌 (Decay)
**操作步骤**:
1. 对比 "过去 28 天" vs "上一周期"。
2. 按点击量差值排序，找到跌幅最大的页面。
3. **行动**: 更新数据年份 (2025 -> 2026)，检查竞争对手是否发布了更好的内容，更新过时信息。

## 📈 模块 B: Google Trends 趋势捕捉 (季度/突发)

### 1. 季节性规划
小檗碱 (Berberine) 等健康话题通常有明显的季节性 (e.g., 新年减肥季)。

**操作步骤**:
1. 访问 Google Trends，输入核心词 "Berberine"。
2. 时间范围选 "过去 5 年" 以观察规律。
3. **行动**: 在流量高峰来临前 **45天** 开始发布或更新相关内容 (e.g., 11月开始准备 "New Year Weight Loss Guide")。

### 2. 突发趋势 (Breakout Trends)
**操作步骤**:
1. 查看 "相关查询" (Related Queries) 中的 "飙升" (Breakout) 词汇。
2. **行动**:
   - 如果与业务高度相关，**24小时内**发布短篇 News/Blog 文章。
   - 在现有高权重文章的开头添加 "最新更新" 说明。

## 📱 模块 C: 社交媒体 Hashtag 监听 (周度)

### 1. 平台监听
重点关注 TikTok, Instagram, Reddit (r/Supplements, r/Berberine)。

**操作步骤**:
1. 搜索核心词 #Berberine。
2. 观察 Top 视频/帖子的共生标签 (Co-occurring Hashtags)。
   - *例子*: 曾出现 #NatureOzempic, #PCOSWeightLoss。
3. **行动**:
   - **内容选题**: 如果 #PCOS 很火，撰写 "Berberine for PCOS" 专题文章。
   - **分发优化**: 在社交媒体分享文章时，带上这些新出现的热门标签。

### 2. 用户问题挖掘 (Reddit/Quora)
**操作步骤**:
1. 查看 Reddit 上关于 Berberine 的最新高赞提问。
2. **行动**: 直接将问题转化为文章的 FAQ 章节，或作为新文章标题 (e.g., "Can I take Berberine with Coffee?").

## 📝 执行清单 (Checklist)

### 月度复盘日 (每月 1 号)
- [ ] 导出 GSC "射程内" 关键词列表
- [ ] 选出 3 篇潜力老文章进行 "Content Refresh"
- [ ] 检查 Google Trends 未来 3 个月的季节性话题
- [ ] 更新 Content Calendar

### 内容更新标准动作 (Refresh SOP)
1. **标题优化**: 加入年份 (2026)，加入新修饰词 (Review, Guide)。
2. **开头重写**: 针对新意图优化第一段。
3. **内容增补**: 增加 1-2 个基于新 Query 的 H2。
4. **死链检查**: 修复失效的外部链接。
5. **发布更新**: 修改 "Last Updated" 时间 (不要修改 URL!)。
6. **Ping**: 重新提交 GSC 索引。
