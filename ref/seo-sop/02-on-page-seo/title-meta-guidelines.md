# Title 与 Meta Description 编写规范

> 标准化的Title和Meta Description是On-Page SEO的基础，直接影响点击率(CTR)和搜索排名

---

## 一、Title 标签规范

### 1.1 基本规则

| 规则 | 要求 | 示例 |
|------|------|------|
| **长度** | 50-60字符（像素宽度≤580px） | ✅ 55字符最佳 |
| **主关键词位置** | 尽量靠前（前30字符内） | ✅ "CT Scanner Price Guide..." |
| **品牌名** | 放最后，用 `|` 或 `-` 分隔 | ✅ "...Guide | MedDevice360" |
| **唯一性** | 每页必须唯一 | ❌ 重复Title |

### 1.2 Title 公式

#### 公式A: 产品/设备页
```
[产品名] - [核心卖点] | [品牌名]
```
**示例**:
- EN: `Siemens SOMATOM go.Top CT Scanner - 64-Slice Premium Performance | MedDevice360`
- ZH: `西门子SOMATOM go.Top CT扫描仪 - 64层高端性能 | MedDevice360`

#### 公式B: 列表/目录页
```
[数量] Best [品类] in [年份] - [价值描述] | [品牌名]
```
**示例**:
- EN: `15 Best CT Scanners in 2025 - Complete Buyer's Guide | MedDevice360`
- ZH: `2025年15款最佳CT扫描仪 - 完整采购指南 | MedDevice360`

#### 公式C: 指南/教程页
```
[How to/Guide] [动作] [对象] - [结果/好处] | [品牌名]
```
**示例**:
- EN: `How to Choose a CT Scanner - Expert Buying Guide for Hospitals | MedDevice360`
- ZH: `如何选择CT扫描仪 - 医院采购专家指南 | MedDevice360`

#### 公式D: 对比页
```
[产品A] vs [产品B] - [年份] [对比维度] Comparison | [品牌名]
```
**示例**:
- EN: `GE Revolution vs Siemens SOMATOM - 2025 CT Scanner Comparison | MedDevice360`
- ZH: `GE Revolution vs 西门子SOMATOM - 2025 CT扫描仪对比 | MedDevice360`

### 1.3 Title 优化技巧

#### ✅ 最佳实践
- 使用数字增加吸引力：`Top 10...`, `5 Ways to...`
- 加入年份显示时效性：`2025 Guide`, `Latest 2025`
- 使用情感词：`Best`, `Ultimate`, `Complete`, `Expert`
- 包含地域词（如适用）：`in China`, `for Southeast Asia`

#### ❌ 常见错误
- 关键词堆砌：`CT Scanner CT Machine CT Equipment Buy CT`
- 过长被截断：`The Complete and Comprehensive Guide to Understanding...`
- 缺少品牌名：容易与竞品混淆
- 使用特殊字符：`★`, `【】` 等可能显示异常

## 二、Meta Description 规范

### 2.1 基本规则

| 规则 | 要求 | 示例 |
|------|------|------|
| **长度** | 150-160字符（像素宽度≤920px） | ✅ 155字符最佳 |
| **主关键词** | 自然融入，出现1-2次 | ✅ 自然描述中包含 |
| **CTA** | 包含行动号召 | ✅ "Get a quote today" |
| **唯一性** | 每页必须唯一 | ❌ 重复Description |

### 2.2 Meta Description 公式

#### 公式A: 产品页
```
[产品简介 + 核心优势]. [关键规格/特点]. [CTA]. [可选: 联系方式]
```
**示例**:
```
EN: Explore the Siemens SOMATOM go.Top 64-slice CT scanner with AI-powered imaging and low-dose technology. Compare specs, view pricing, and request a quote today.

ZH: 探索西门子SOMATOM go.Top 64层CT扫描仪，配备AI驱动成像和低剂量技术。对比参数、查看价格，立即获取报价。
```

#### 公式B: 列表页
```
[内容价值描述]. [包含内容概览]. [CTA].
```
**示例**:
```
EN: Compare the top 15 CT scanners from leading manufacturers including GE, Siemens, and Philips. Detailed specs, pricing, and expert reviews. Find your ideal scanner today.

ZH: 对比来自GE、西门子、飞利浦等领先制造商的15款顶级CT扫描仪。详细规格、价格和专家评测。立即找到您的理想设备。
```

#### 公式C: 指南页
```
[问题/需求描述]. [解决方案概述]. [CTA].
```
**示例**:
```
EN: Planning to buy a CT scanner for your hospital? Our comprehensive guide covers selection criteria, pricing factors, and negotiation tips. Download free checklist now.

ZH: 计划为医院采购CT扫描仪？我们的完整指南涵盖选型标准、价格因素和谈判技巧。立即下载免费清单。
```

### 2.3 高转化CTA词汇库

#### 英文CTA
| 类型 | 示例 |
|------|------|
| 获取报价 | Get a quote, Request pricing, Get your quote |
| 了解更多 | Learn more, Discover, Explore |
| 立即行动 | Start now, Try free, Book demo |
| 下载资源 | Download free, Get the guide, Access now |
| 对比选择 | Compare now, Find your match, See options |

| 类型 | 示例 |

| 对比选择 | 立即对比, 找到匹配, 查看选项 |

## 三、多语言适配

### 3.1 英文 → 中文转换注意事项

| 问题 | 解决方案 |
|------|----------|
| 中文字符占更多像素 | 中文Title控制在25-30个汉字 |
| 直译不自然 | 保持意思一致但重新组织语言 |
| CTA文化差异 | 使用本地化表达 |

### 3.2 hreflang 配合

确保Title/Meta与hreflang标签配合：
```html
<link rel="alternate" hreflang="en" href="https://example.com/ct-scanners" />
<link rel="alternate" hreflang="zh" href="https://example.com/zh/ct-scanners" />
```


### 4.2 动态Title生成

```tsx
// 产品页动态Title
const generateProductTitle = (product: Device) => {
  const brand = product.manufacturer_name;
  const model = product.name_en;
  const type = product.type;
  return `${brand} ${model} ${type} - Specifications & Pricing | MedDevice360`;
};

// 列表页动态Title
const generateListTitle = (count: number, category: string, year: number) => {
  return `${count} Best ${category} in ${year} - Complete Buyer's Guide | MedDevice360`;
};
```

## 五、审计清单

### 发布前检查

- [ ] Title长度在50-60字符之间
- [ ] 主关键词在Title前30字符内
- [ ] Title包含品牌名
- [ ] Meta Description长度在150-160字符之间
- [ ] Meta Description包含CTA
- [ ] Title和Description均唯一（与其他页面不重复）
- [ ] 中英文版本均已配置
- [ ] 结构化数据中的name/description与页面一致

### 批量审计工具

使用Screaming Frog导出所有页面的Title/Meta：
1. 爬取网站
2. 导出 `Page Titles` 和 `Meta Descriptions` 报告
3. 检查：长度、重复、缺失
4. 标记需要优化的页面

## 六、A/B测试建议

### 可测试元素
- Title中关键词位置（前置 vs 中置）
- 是否包含年份
- CTA用词选择
- 是否使用数字
- 情感词使用（Best vs Top vs Leading）

### 测试方法
1. 选择流量>1000/月的页面
2. 在GSC中监控CTR变化
3. 测试周期：2-4周
4. 记录变化并迭代
