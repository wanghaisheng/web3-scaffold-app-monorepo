## 🔄 内容生产流程

### 阶段1: 规划（D-14）

```markdown
检查清单:
- [ ] 确定内容主题和角度
- [ ] 关键词研究完成
- [ ] 竞品内容分析
- [ ] 确定目标字数
- [ ] 分配负责人
- [ ] 设定截止日期
```

### 阶段2: 创作（D-14 to D-7）

```markdown
检查清单:
- [ ] 创建内容大纲
- [ ] 大纲审核通过
- [ ] 撰写初稿
- [ ] 添加数据/引用
- [ ] 制作配图/图表
- [ ] 初稿自检
```

### 阶段3: 审核（D-7 to D-3）

```markdown
检查清单:
- [ ] 内容准确性审核
- [ ] SEO优化检查
- [ ] 语法/拼写检查
- [ ] 内链添加
- [ ] CTA设置
- [ ] Schema数据准备
```

### 阶段4: 发布（D-3 to D-Day）

```markdown
检查清单:
- [ ] 上传到CMS
- [ ] 元数据设置
- [ ] 图片Alt文本
- [ ] URL设置
- [ ] 预览测试
- [ ] 正式发布
- [ ] 社交媒体分享
```
