# 内容更新SOP

> "Fresh content signals relevance to search engines." — 内容更新是保持排名竞争力的关键。

## 📋 内容更新策略概览

### 更新优先级矩阵

| 内容类型 | 更新频率 | 优先级 | 触发条件 |
|---------|---------|-------|---------|
| 核心Money Pages | 每3个月 | P0 | 排名下降/数据过期 |
| 产品对比页 | 每6个月 | P0 | 新产品发布 |
| 市场分析 | 每年 | P1 | 年度数据更新 |
| 技术指南 | 每12个月 | P1 | 技术/法规变化 |
| 案例研究 | 按需 | P2 | 客户更新 |
| 常青内容 | 每18个月 | P2 | 链接/流量下降 |

### 更新触发信号

```
🚨 立即更新信号:
- 关键页面排名下降>5位
- 内容包含过时数据/价格
- 竞品发布更好内容
- 用户反馈内容错误
- 法规/政策变化

⏰ 定期检查信号:
- 页面流量持续下滑
- 跳出率上升
- 停留时间减少
- 反向链接减少
```

## 🔄 内容审计流程

### 季度内容审计

#### 第1步: 内容清单导出

```sql
-- 概念性SQL，实际使用GSC/GA导出
SELECT 
  url,
  title,
  publish_date,
  last_updated,
  organic_traffic_90d,
  avg_position,
  impressions,
  clicks
FROM content_pages
ORDER BY organic_traffic_90d DESC
```

#### 第2步: 内容分类

```
📊 内容健康度分类:

🟢 健康 (Keep):
- 流量稳定或增长
- 排名Top 10
- 转化率达标
→ 动作: 监控，无需立即更新

🟡 需关注 (Improve):
- 流量轻微下滑(<20%)
- 排名11-30位
- 内容略显过时
→ 动作: 安排3个月内更新

🔴 紧急 (Update Now):
- 流量大幅下滑(>30%)
- 排名30+或跌出首页
- 内容严重过时
→ 动作: 立即安排更新

⚫ 淘汰 (Consolidate/Remove):
- 几乎无流量
- 与其他内容重复
- 主题不再相关
→ 动作: 合并或301重定向
```

#### 第3步: 优先级排序

```
更新优先级公式:
Priority Score = (Traffic Potential × Content Gap) / Update Effort

考虑因素:
- 关键词搜索量
- 当前排名位置
- 竞品内容质量
- 更新工作量
```

## ✏️ 内容更新执行

### 更新类型与范围

#### 轻度更新 (Light Refresh)
**工作量**: 1-2小时  
**适用**: 排名稳定但需微调

```markdown
更新内容:
- [ ] 更新日期/年份引用
- [ ] 修正小错误
- [ ] 更新统计数据
- [ ] 添加1-2个新内链
- [ ] 优化Meta Description
- [ ] 更新Last Modified日期
```

#### 中度更新 (Content Enhancement)
**工作量**: 4-8小时  
**适用**: 排名下滑需要提升

```markdown
更新内容:
- [ ] 上述轻度更新全部
- [ ] 添加新章节（500-1000字）
- [ ] 更新/添加图片和图表
- [ ] 增强结构化数据
- [ ] 添加FAQ部分
- [ ] 改善内链策略
- [ ] 优化H标签结构
- [ ] 添加专家引用
```

#### 重度更新 (Major Rewrite)
**工作量**: 8-16小时  
**适用**: 内容严重过时或竞品大幅领先

```markdown
更新内容:
- [ ] 保留URL，完全重写内容
- [ ] 重新进行关键词研究
- [ ] 竞品内容分析
- [ ] 新的内容大纲
- [ ] 2000+字新内容
- [ ] 全新图片和可视化
- [ ] 视频/互动元素
- [ ] 更新所有内外链
- [ ] 重新提交GSC索引
```

## 📝 更新检查清单

### 更新前准备

```markdown
- [ ] 导出当前页面数据（排名、流量、链接）
- [ ] 分析排名前10的竞品
- [ ] 识别内容差距
- [ ] 确定更新范围
- [ ] 准备更新素材（数据、图片）
```

### 内容更新要点

```markdown
SEO优化检查:
- [ ] Title包含目标关键词（前60字符）
- [ ] Meta Description更新（155字符内）
- [ ] H1仅一个且包含关键词
- [ ] H2/H3结构清晰
- [ ] 关键词自然分布
- [ ] 图片Alt文本更新
- [ ] 内链至少3个
- [ ] 外链引用有效

内容质量检查:
- [ ] 数据和统计是最新的
- [ ] 产品信息准确
- [ ] 价格信息更新
- [ ] 删除过时引用
- [ ] 添加最新趋势
- [ ] 语法和拼写检查
```

### 更新后操作

```markdown
- [ ] 更新Last Modified日期
- [ ] 更新Schema dateModified
- [ ] 在GSC请求重新索引
- [ ] 更新Sitemap
- [ ] 社交媒体重新分享
- [ ] 通知邮件列表（可选）
- [ ] 记录更新到追踪表
```
